РУКОВОДСТВО ПО УСТАНОВКЕ

Для работы с консольной версией Геометрии нужна установка
Java Runtime Environment (JRE) версии 1.5 или выше. Если JRE на компьютере
еще не установлена, ее можно скачать со следующего адреса:

http://java.com

JRE должна соответствовать операционной системе, будь это Windows,
Mac, Linux или Solaris.

Если JRE установлена, но версия стара (предшествует 1.5), ее необходимо
обновить. Скачать новейшую версию JRE можно с адреса выше.

Исполнительная программа JRE должна находиться в средовой переменной PATH.
Чтобы это проверить, открой командное окно (Windows) или терминал
(Mac, Linux, Solaris) и введи:

java -version

Должна вывеситься установленная версия JRE. Если нет, следуй указаниям
на следующей страничке:

http://java.com/en/download/help/path.xml

Чтобы запустить Геометрию:

Пользователь Windows:
Щелкни дважды по файлу geometria.bat.

Пользователь Mac, Linux, Solaris:
Открой терминал, введи команду:

./geometria.sh

Pуководство пользователя находится по адресу:

http://geocentral.net/geometria/ru
