GUIA RAPIDA

La versión de escritorio de GEOMETRIA requiere tener instalado Java Runtime 
Environment 1.5 o superior en el equipo de trabajo. De no contar con JRE ya instalado,
puede descargarse de:

http://java.com

Conviene asegurarse de descargar el compendio JRE adecuado para el sistema operativo
de destino, sea que se tratase de Windows, Mac, Linux o Solaris.

Aviso: De contar con una versión previa de JRE, será necesario actualizarla a la
versión 1.5 o superior. Se puede descargar la versiçon más reciente de JRE de la
direcciçon ya mencionada.

Debe corroborarse que el JRE ejecutable está ubicado en el paso adecuado. PAra 
controlas que el JRE estça en el paso, debe abrirse la ventana de comandos 
(Windows) o una terminal (Mac, Linux, Solaris) y anotar:

java -version

Debe aparecer la versiçon de JRE en uso. De no ser así, deben seguirse las 
indicaciones que se despliegan en:

http://java.com/en/download/help/path.xml

Para ejecutar GEOMETRIA:

Versión Windows:
Doble clic en el archivo geometria.bat.

Usuarios de Mac, Linux, Solaris:
Abrir una terminal, anotar bajo el cursor de comando:

./geometria.sh

Una Guçia de Uso completa estça disponible en:

http://geocentral.net/geometria/en
