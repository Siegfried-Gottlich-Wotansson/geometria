GUIDE D'INSTALLATION

La version poste de travail (local) de GEOMETRIA nécessite le
Java Runtime Environment 1.5 ou plus installé sur votre ordinateur.
Si JRE n'est pas déjà installé, vous pouvez le télécharger de: 

http://java.com

Assurez-vous de télécharger la version de JRE compatible avec votre
système d'opération, que ce soit Windows, Mac, Linux ou Solaris. 

Note: Si vous disposez d'une version antérieure de JRE, vous devrez
faire la mise-à-jour à la version 1.5 ou plus. Vous pouvez télécharger
la dernière version de JRE du site mentionné auparavant.

Assurez-vous que l'exécutable JRE se retrouve dans vos variables
d'environnement. Afin de vérifier si JRE est fonctionnel, ouvrez
une fenêtre de commandes (Windows) ou un terminal (Mac, Linux, Solaris)
et saisissez: 

java -version

Vous devez voir la version de JRE que vous utilisez. Si rien
ne se produit, suivez les instructions à:

http://java.com/en/download/help/path.xml

Afin de démarrer GEOMETRIA:

Utilisateurs Windows:
Double-cliquez le fichier geometria.bat.

Utilisateurs Mac, Linux, Solaris:
Ouvrez un terminal, et entrez à l'invite de commande: 

./geometria.sh

Un Guide complet se trouve en-ligne à:

http://geocentral.net/geometria/fr
